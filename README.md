### About Theme

#### this theme have Four color theme

- WD Light
- WD Dark Modern
- WD Dark Cool

Notes:

- If Italic style not work, please install "Fira Code iScript" font
- And add this in setting.json -
  - "editor.fontFamily": "Fira Code iScript",
  - "editor.fontLigatures": true,
